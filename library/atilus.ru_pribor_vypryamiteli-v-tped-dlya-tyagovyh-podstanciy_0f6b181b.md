Выпрямители В-ТПЕД для тяговых подстанций предназначены для питания контактной сети метрополитена и тяговых подстанций городского электрифицированного транспорта.

Обозначение
-----------

Выпрямители В-ТПЕД обозначаются так: В-ТПЕД- Х1- Х2- Х3Х4Х5- Х6Х7, где

*   В-ТПЕД - выпрямитель трехфазного тока в постоянный ток с естественным охлаждением, диодный;
*   Х1 - номинальный выходной ток, А;
*   Х2 - номинальное выходное напряжение, В;
*   Х3 - модификация исполнения: 
    *   0 - трехфазная мостовая схема выпрямления,
    *   1 - шестифазная схема выпрямления с уравнительным реактором;

*   Х4 - исполнение по наличию питающего силового трансформатора: 
    *   0 - без трансформатора,
    *   1 - с трансформатором;

*   Х5 - исполнение по наличию выходного выключателя. 
    *   0 - без выключателя,
    *   1 - с выключателем;

*   Х6 - вид климатического исполнения по ГОСТ 15150;
*   Х7 - категория размещения по ГОСТ 15150.

Конструктивное исполнение
-------------------------

Выпрямители В-ТПЕД выполнен в виде шкафа двухстороннего обслуживания с возможностью доступа для подключения к силовой части и ремонта. Исполнение подключения шин- стандартное.

Силовая часть выполнена на силовых полупроводниковых приборах. Охлаждение выпрямителей - естественное воздушное. Сборка силовой части производится по технологии необслуживаемых контактных соединений, благодаря этому увеличивается межремонтный период обслуживания изделий, уменьшаются расходы на техническое обслуживание, повышается пожаробезопасность и надежность выпрямителя.

Для защиты силовых приборов от перенапряжений со стороны переменного тока применены варисторы. Для снижения коммутационных перенапряжений диоды зашунтированы RC цепочками. Выпрямитель В-ТПЕД имеет аппаратуру управления контроля и сигнализации, обеспечивающую режимы управления масляным выключателем МВ и быстродействующим выключателем БВ.
